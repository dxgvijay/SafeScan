Security Guarantees

Every execution runs in a fireproof container. Here's what's enforced.

🔒

Network Isolated

No internet access from the sandbox. All network requests are blocked at the kernel level.

⏱

10s Time Limit

Automatic kill after 10 seconds of execution. Prevents runaway processes and resource abuse.

💾

64MB RAM Cap

Memory usage is strictly limited to 64MB per execution. No swapping, no leaks.

🚫

No System Access

Restricted file system — read-only system paths. Can only write to temporary scratch space.

🧹

Auto Cleanup

All temp files and containers are destroyed immediately after execution completes.

👁

Full Logging

Every execution is logged: command, output, exit code, duration, and resource usage.
