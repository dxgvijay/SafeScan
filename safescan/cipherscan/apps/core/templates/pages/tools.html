How each tool works

Phishing Analyzer
  • 01 Paste email headers or upload a suspicious email as .eml or .msg file.
  • 02 The engine parses headers, checks SPF/DKIM/DMARC authentication, and extracts all links.
  • 03 Each extracted URL is scanned against phishing databases and analyzed for spoofing patterns.
  • 04 You receive a detailed report with threat score, verdict, and recommended actions.
URL Scanner
  • 01 Enter a URL or paste a list of URLs for batch scanning.
  • 02 The scanner fetches the page content, follows redirects, and extracts embedded resources.
  • 03 Results are cross-referenced against 70+ threat intelligence vendors and databases.
  • 04 View a comprehensive report including WHOIS, SSL status, domain reputation, and threat verdict.
File Scanner
  • 01 Upload a file up to 256 MB. Supports PDF, EXE, DOCX, ZIP, APK, BAT, and 50+ other formats.
  • 02 The file is analyzed by multiple anti-virus engines and YARA rules simultaneously.
  • 03 Behavioral analysis detects suspicious patterns like packers, obfuscation, and known malware signatures.
  • 04 Download a detailed report with per-engine results, IoCs, and severity classification.
Code Sandbox
  • 01 Paste or upload code in Python, JavaScript, C++, Bash, Go, Rust, or Java.
  • 02 The code runs in a secure, isolated container with network monitoring and filesystem tracking.
  • 03 All system calls, network connections, and file modifications are recorded for analysis.
  • 04 Review the execution trace, identify malicious behavior, and get a safety verdict.
IP & Domain Intel
  • 01 Enter an IP address or domain name to begin investigation.
  • 02 The system fetches geolocation, ASN, reverse DNS, and WHOIS data.
  • 03 Results are checked against 30+ blacklist databases and abuse registries.
  • 04 View open ports, associated domains, and a comprehensive risk score.
Reports & History
  • 01 Browse your complete scan history with filters by date, type, and severity.
  • 02 Click any scan to view the full detailed report with all findings.
  • 03 Export reports as PDF or share them with your team via a secure link.
  • 04 Access programmatic results and trigger scans via our REST API with your API key.

  -H "Authorization: Bearer cs_api_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "url",
    "target": "https://example.com",
